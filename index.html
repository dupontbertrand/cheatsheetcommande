CheatSheet Commandes Ubuntu

  • man + commande : Ouvre le MANuel de la commande (L'aide)
  • cd + repertoire : Se déplacer (cd .. : Se déplace au rep parent et cd / : Se déplace à la racine)
  • ls : LiSter
  • ls -a : LiSter -All
  • ls -l : LiSter -Long (Types/Droits/User/Taille/Date/Heure/Nom)
  • pwd : Repertoire courant
  • cp : CoPie
  • mv : MoVe/Déplacer ou renommer (Dans ce dernier cas : mv exemple.html index.html = Renomme exemple.html en index.html )
  • rm : Suppression (ReMove)
  • mkdir : Création repertoire dans le rep courant ( -p permet de créer les rep parents si ils n'existent pas )
  • rmdir : Suppression repertoire
  • apt-get moo : Une vache ! (Trés important)
  • ren : REName
  • CTRL + L : Clear (ou taper clear)
  • atom . : Permet d'ouvrir dans atom tous les fichiers du rep courant
  • cat > Exemple Test CTRL + D Va créer un fichier Exemple dans le rep courant avec Test comme texte
  • ls > Test Va lister le rep courant dans un fichier Test
